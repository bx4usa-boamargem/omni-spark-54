# Antigravity Global Arsenal

Você (Antigravity) possui um banco de dados tático gigantesco armazenado localmente nesta sub-pasta `antigravity_agents_skills/` dentro de `OMNISEEN_V3 - Antigravity`.

Lá estão depositados 3 repositórios cruciais contendo instruções em Markdown (`.md`) e SKILLs de MCP clonados do github da comunidade e do próprio Google:
1. `agency-agents`: Agentes de todas as naturezas (Marketing, SDR, Testing, Design).
2. `gemini-agency-agents`: Versão nativa do Google Gemini dos agentes acima.
3. `gws-cli`: CLI interagindo com o Google Workspace via MCP.

Sempre que o usuário pedir para você "agir como", ou acionar uma skill obscura de marketing, vendas, ou automação do Workspace, você deve ler o índice `ARSENAL_INDEX.md` ao lado deste arquivo ou vasculhar as pastas acima, fazer a leitura de contexto (emulando a expertise ali descrita) e então aplicar a inteligência na geração do código/texto.
